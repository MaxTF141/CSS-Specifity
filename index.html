CSS Specifity

It's a method for resolving conflicts when selectors with different style rules applied to the same element. Specifity only applies when when multiplied declarations target the same element.

How does specifity hierarchy work?

Specifity involves creating scores or compare selectors. It uses a four-part specifity hierarchy to determine a selector's specifity score.

It goes as follows:

  • Inline style selector selectors are style rules defined in a HTML document using the < style > attribute. These selectors are given a score of 1000.
  • ID selector selectors have a specifity score of 100.
  • Class, Pseudo-class, attribute selectors have a score of 10.
  • Element and pseudo-element selectors have a specifity score of 1.

NB! The *Universal selector have a specifity score of 0. It's style rules will be overwritten whenever there is conflicting selectors.
